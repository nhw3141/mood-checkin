Cloud sync upgrade path

This app stores entries in IndexedDB by default. To upgrade to cloud sync later:
  1. Add login (email/password or Google).
  2. Replace the DataStore functions with Firebase/Supabase calls.
  3. Keep the same entry format (timestamp, emotion, note, slot time).

Data management

Be careful: deleting clears data in this browser only.
